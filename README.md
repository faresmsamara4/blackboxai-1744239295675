# Disease Treatment Guide

## Project Overview
The **Disease Treatment Guide** is a web-based application designed to provide valuable information on common diseases and their treatment options. This guide offers insights into home treatments, professional care, and natural remedies for various conditions. It aims to empower users with knowledge about health management in an easily accessible format.

## Installation
To use the **Disease Treatment Guide**, simply download or clone the repository and open the `disease_treatments.html` file in any modern web browser.

### Steps to Install:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/disease-treatment-guide.git
   ```
2. Navigate to the project directory:
   ```bash
   cd disease-treatment-guide
   ```
3. Open the `disease_treatments.html` file in a web browser.

## Usage
Once the HTML file is open in your browser, you will see a user-friendly interface displaying informative boxes for different diseases. Each box contains:
- Disease name
- Home treatment options
- Professional treatment options
- Natural remedies

Simply scroll through the list of diseases to find the relevant information.

## Features
- **Responsive Design:** Utilizes Tailwind CSS for a modern and responsive layout that looks good on both desktop and mobile devices.
- **Comprehensive Coverage:** Includes treatments for various conditions such as Common Cold, Influenza, Hypertension, Diabetes, Acid Reflux, and Eczema.
- **User-Friendly Interface:** The design is easy to navigate, making it accessible for users of all ages.
- **Stylish Template:** Leverages Font Awesome for intuitive icons next to treatment types (Home, Professional, and Natural).

## Dependencies
The project relies on the following external libraries:
- [Tailwind CSS](https://tailwindcss.com/) for styling and layout.
- [Font Awesome](https://fontawesome.com/) for accessible icons.

These libraries are referenced via CDN in the HTML file.
